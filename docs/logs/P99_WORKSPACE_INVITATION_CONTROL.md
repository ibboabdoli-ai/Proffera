# P99 — Workspace invitation control

## Outcome

Workspace Owners can now manage invitations that are still pending from **Inställningar → Team och behörigheter**.

## Available actions

- View the invited name, email, role, and Stockholm expiry time.
- Send a new email with a fresh one-time link. The prior link is immediately invalidated.
- Revoke an unused invitation. A revoked link can no longer create an account or membership.

## Safeguards

- The pending-invitation list and actions are Owner-only.
- Each action is scoped to the current workspace.
- The Service AI Chat and dashboard AI assistant remain unchanged.
