# P102 — Restricted dashboard copy

## Outcome

The restricted dashboard now matches the access level of an invited Staff user:

- Settings is not offered as a shortcut when the user cannot open it.
- The hero and supporting copy explain that workspace modules must be activated.
- Available links are labeled as the user's available parts, not as a full customer-workflow dashboard.

## Scope

- Presentation and navigation eligibility only.
- No role, plan, customer, booking, workspace, or AI data is changed.
